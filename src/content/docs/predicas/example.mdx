# Ejemplo con Markmap

import Markmap from '@components/Markmap.astro';

El siguiente ejemplo convierte un bloque de Markdown en un mapa mental interactivo gracias a **Markmap**. Modifica el contenido para adaptarlo a tus enseñanzas.

```md
# Serie: Fundamentos de la Fe

## La Salvación
- Por gracia mediante la fe
- No por obras
- Seguridad eterna

## Crecimiento Espiritual
- Oración diaria
- Estudio bíblico
- Comunión con la iglesia
- Servicio

## Misión
- Compartir el evangelio
- Hacer discípulos
```

  content={`
